# Guia de Templates de Banner — Instituto de Mãos Dadas

Este guia define as regras para criação de banners digitais para as diferentes frentes de trabalho.

## 1. Estrutura Geral
- **Headline**: Anton (Uppercase)
- **Apoio**: Inter (Regular)
- **Tag**: Roboto Bold (Background colorido)

## 2. Tipos de Banner

### A. Institucional (Padrão)
- **Fundo**: Gradiente de Roxo para Roxo Escuro
- **Acento**: Laranja (`#FF9E00`)
- **Uso**: Avisos gerais, comunicados oficiais.

### B. Comunidade Consciente
- **Fundo**: Azul Slate (`#1E3A5F`)
- **Efeito**: Glassmorphism obrigatório em caixas de dados.
- **Uso**: Relatórios de progresso, dashboards comunitários.

### C. Mobilização (Lambe-lambe)
- **Fundo**: Branco Texturizado (Papel)
- **Efeito**: Tipografia Anton com `mix-blend-mode: multiply`.
- **Uso**: Chamadas para mutirões, protestos, denúncias.
